# Book Recommendation Engine

This repository contains a machine learning recommendation system built as a project for the Machine Learning with Python Certification by freeCodeCamp.

## Project Overview
The goal of this exercise was to build a K-Nearest Neighbors (KNN) algorithm capable of recommending similar books based on a large dataset of user ratings. It demonstrates the practical implementation of data cleaning, sparse matrix transformations, and instance-based machine learning algorithms.

## Files in this Repository
* `fcc_book_recommendation_knn.ipynb`: The Google Colab notebook containing the data filtering, model building, and evaluation steps required to pass the certification testing script.

## Technology Stack
* Python
* Scikit-Learn (`NearestNeighbors`)
* Pandas & SciPy
